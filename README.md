# Flask API Boilerplate

## Include
- Flask
- Docker
- Docker-Compose
- OpenQL
- Swagger (coming soon)
- Redis (coming soon)

## Instruction
1. Clone this repository
    ```
    git clone git@github.com:tetsu/flask-api-boilerplate.git
    ```
1. run 'docker-compose build'
    ```
    docker-compose build
    ```
1. run 'docker-compose up'
    ```
    docker-compose up
    ```

## Endpoints
1. Top Page: /
1. OpenQL: /openql
1. REST API: /restapi
