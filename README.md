# Flask API Boilerplate

## Include
- Flask
- Docker
- Docker-Compose
- Swagger (coming soon)
- Redis (coming soon)
- OpenQL (coming soon)

## Instruction
1. Clone this repository
    ```
    git clone git@github.com:tetsu/flask-api-boilerplate.git
    ```
1. run 'docker-compose build'
    ```
    docker-compose build
    ```
1. run 'docker-compose up'
    ```
    docker-compose up
    ```
